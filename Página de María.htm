About HTML Preprocessors

HTML preprocessors can make writing HTML more powerful or convenient. For instance, Markdown is designed to be easier to write and read for text documents and you could write a loop in Pug.

Learn more · Versions

Adding Classes

In CodePen, whatever you write in the HTML editor is what goes within the <body> tags in a basic HTML5 template. So you don't have access to higher-up elements like the <html> tag. If you want to add classes there that can affect the whole document, this is the place to do it.

About the <head>

In CodePen, whatever you write in the HTML editor is what goes within the <body> tags in a basic HTML5 template. If you need things in the <head> of the document, put that code here.

CSS

About CSS Preprocessors

CSS preprocessors help make authoring CSS easier. All of them offer things like variables and mixins to provide convenient abstractions.

Learn more · Versions

Need an add-on?

CSS Base

About CSS Base

It's a common practice to apply CSS to a page that styles elements such that they are consistent across all browsers. We offer two of the most popular choices: normalize.css and a reset. Or, choose Neither and nothing will be applied.

Vendor Prefixing

About Vendor Prefixing

To get the best cross-browser support, it is a common practice to apply vendor prefixes to CSS properties and values that require them to work. For instance -webkit- or -moz-.

We offer two popular choices: Autoprefixer (which processes your CSS server-side) and -prefix-free (which applies prefixes via a script, client-side).

Add External Stylesheets/Pens

Any URLs added here will be added as <link>s in order, and before the CSS in the editor. You can use the CSS from another Pen by using its URL and the proper URL extension.

Powered by
About External Resources

You can apply CSS to your Pen from any stylesheet on the web. Just put a URL to it here and we'll apply it, in the order you have them, before the CSS in the Pen itself.

You can also link to another Pen here (use the .css URL Extension) and we'll pull the CSS from that Pen and include it. If it's using a matching preprocessor, use the appropriate URL Extension and we'll combine the code before preprocessing, so you can use the linked Pen as a true dependency.

<h2>María Jesús Almazor: Una líder en telecomunicaciones y tecnología</h2>

<p>María Jesús Almazor Marsal (Barcelona, 1970) es una destacada <b>ingeniera de telecomunicaciones y ejecutiva española</b>, reconocida por su liderazgo en el sector tecnológico y su <b>compromiso con la diversidad y la transformación digital</b>.​

<br>María Jesús Almazor es una destacada ejecutiva española en el sector de las telecomunicaciones y la tecnología. Nacida en Barcelona, es Ingeniera de Telecomunicaciones por la<b> UPC</b> y ha desarrollado toda su carrera profesional en <b>Telefónica desde 1994</b>. Comenzó como optimizadora de la red radio en Telefónica Móviles España y rápidamente asumió responsabilidades en el despliegue de cobertura móvil en Cataluña. A lo largo de su trayectoria, ha ocupado diversos cargos de liderazgo, incluyendo <b>Directora de Operaciones del Territorio Norte y Sur de Telefónica España, y Consejera Delegada de Telefónica España</b> en 2018. En 2021, fue nombrada <b>CEO de la división de Ciberseguridad y Cloud de Telefónica Tech</b>, el holding de negocios digitales de Telefónica .

<br>Además de su exitosa carrera profesional, Almazor ha sido reconocida por su liderazgo y contribución al progreso económico y social. En 2021, recibió el premio <b>‘Mujer y Empresa Capital’</b> durante los Premios Andalucía Capital de Onda Cero, destacando la labor desarrollada por Telefónica y su impacto en la región . También fue elegida como una de las <b>‘Top 100 Mujeres Líderes en España’</b> en la categoría de alta dirección, un reconocimiento impulsado por la revista Mujeres&cia para visibilizar el talento femenino en diversos ámbitos .

<br>María Jesús Almazor es una figura clave en la transformación digital de Telefónica y un referente del liderazgo femenino en el sector tecnológico.</p>
    
<h3>Empresas en las que ha trabajado María.</h3>
    
<p>María Jesús Almazor ha desarrollado la mayor parte de su carrera profesional en Telefónica, una de las compañías más grandes en telecomunicaciones y tecnología a nivel global. Su trayectoria en la empresa abarca diversos roles de liderazgo, desde sus inicios como optimizadora de redes hasta su nombramiento como <b>CEO de la división de Ciberseguridad y Cloud en Telefónica Tech</b>, el brazo de negocios digitales de Telefónica.
